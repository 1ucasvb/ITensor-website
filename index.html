ITensor—Intelligent Tensor—is an efficient and flexible C++ library for implementing tensor product wavefunction calculations.
Features include:

  • A complete DMRG code
  • Quantum number conserving tensors
  • Efficient matrix product state types
  • Complex numbers (handled lazily: no efficiency loss if real)

Multiplying ITensors is as easy as multiplying scalars: matching indices automatically contract, making it simple to transcribe tensor network diagrams into correct, efficient code.

Code Samples

Perform a DMRG Calculation

  • SpinOne model(100); //define Hilbert space of 100 spin-lattice sites
  • MPO H = Heisenberg(model); //pre-defined Hamiltonian; easy to write your own
  • MPS psi(model);
  •  
  • Sweeps sweeps(5); //perform 5 sweeps of DMRG
  • sweeps.maxm() = 50, 50, 100, 100, 200; //max number of states kept each sweep
  •  
  • dmrg(psi,H,sweeps); //run the DMRG algorithm
  • Real energy = psiHphi(psi,H,psi); //can analyze wavefunction *after* DMRG

Multiply Two ITensors

  • Index a("a",2), b("b",2), c("c",2);
  • ITensor Z(a,b), X(b,c);
  • commaInit(Z,a,b) << 1, 0,
                        0,-1;
  • commaInit(X,b,c) << 0, 1,
                        1, 0;
  • ITensor R = Z * X;
  • Print( R(a(1),c(2)) );
  •   //Prints: R(a(1),c(2)) = 1
  • Print( R(a(2),c(1)) );
  •   //Prints: R(a(1),c(2)) = -1
