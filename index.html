ITensor is a C++ library for creating efficient and flexible physics simulations using tensor product wavefunctions.

Multiplying ITensors is just as easy as multiplying scalars; matching indices automatically find each other and contract, making it simple to convert tensor network diagrams into correct, efficient code.

Code Samples

Perform a DMRG Calculation

  • int N = 100;
  • SpinOne model(N);
  • MPO H = Heisenberg(model);
  • MPS psi(model);
  • Real energy = dmrg(psi,H);

Multiply Two ITensors

  • Index a("a",2), b("b",2), c("c",2);
  • ITensor Z(a,b), X(b,c);
  • commaInit(Z) << 1, 0,
                    0,-1;
  • commaInit(X) << 0, 1,
                    1, 0;
  • ITensor R = Z * X;
  • Print( R(a(1),c(2)) );   //Prints R(a(1),c(2)) = 1
  • Print( R(a(2),c(1)) );   //Prints R(a(1),c(2)) = -1
