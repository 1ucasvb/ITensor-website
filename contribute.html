Become a Contributor

Steps to becoming a library contributor:

  1. Create a github account, then fork our code repository.

    (Help on forking a repo.)

  2. Make your edits and commit them. It may be helpful to make multiple commits outlining the changes at each step.

  3. Push the changes to your personal fork (usually just a git push), then send us a pull request.
